=============
Instructions:
=============
1. Download XAMPP from Link given blow.
2. Install XAMPP into your PC
3. Copy Folder "PrinteX" to /Applications/XAMPP/xamppfiles/htdocs\
4. Open XAMPP Control Panel
5. Start Apache and MySQL in XAMPP Control Panel
6. Goto PHPMyAdmin page in your browser. (Link given blow).
7. Create new database named PrinteX_db.
8. Goto database and import given database from "PrinteX/database" folder
9. Import database named  "PrinteX_db.sql"
10. Login on login page and interact with website.
Note: Login Id: Admin, Pass: admin123
